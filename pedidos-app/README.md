# Sistema de Pedidos

Aplicación JavaFX para gestionar pedidos y despachos de productos.

## Requisitos

- Java 11 o superior
- Maven

## Cómo ejecutar

1. Navegar al directorio del proyecto: `cd pedidos-app`
2. Compilar y ejecutar: `mvn clean compile javafx:run`

## Funcionalidades

- Registrar un pedido con cliente y productos
- Consultar información y estado de un pedido
- Despachar un pedido

Estados del pedido: SOLICITADO -> REGISTRADO -> DESPACHADO
